# Stockpile SOP

## Purpose

Define lot-level control between receiving and processing.

## Rules

- Each stockpile lot must identify case linkage, location code, material summary, quantity, unit and status.
- Lot movement or merge/split activity must be auditable.
- Certificate eligibility depends on reliable lot and disposition evidence.

## Records

- `stockpile_lots`
- `processing_batches`
- `case_events`
