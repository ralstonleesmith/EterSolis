# Processing SOP

## Purpose

Define processing and disposition evidence for recovery, repurpose, destruction, treatment or disposal pathways.

## Rules

- Processing batches must capture inputs, outputs, route, status and completion time.
- Disposition evidence is required before certificate approval.
- Unsupported, restricted or uncertain materials remain in technical review.

## Records

- `processing_batches`
- `disposition_records`
- `case_files`
- `case_events`
