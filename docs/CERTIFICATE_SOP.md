# Certificate SOP

## Purpose

Define certificate issuance controls for repurpose and destruction certificates.

## Rules

- Certificates require service request/case linkage, receiving or disposition evidence, admin approval, certificate number, verification hash and audit event.
- Public verification shows validity, certificate number/type, issue date, department, material summary, quantity summary and verification hash only.
- Internal notes, pricing, documents, photos and logistics are never exposed publicly.
