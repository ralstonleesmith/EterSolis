# Upgrade: site polish & accessibility fixes — 2026-06-14

Summary
- Replaced inline SVG brand usage with approved mark PNG (`public/media/etersolis-mark.png`) where necessary.
- Fixed `image-redundant-alt` by setting decorative logo `alt=""` in `src/components/brand/EterSolisLogo.tsx`.

- Stabilized Playwright a11y scans by ensuring deterministic theme (light) during runs and adding small settle time.

- `src/app/globals.css` — variables and targeted utility overrides for `.text-coal/60`, `.text-coal/70`, overlay helpers.
- `src/components/brand/EterSolisLogo.tsx` — decorative `alt` for full logo.
- `tests/e2e/a11y.spec.ts` — a11y test runner improvements for determinism.

How to verify locally

1. Install dev dependencies:

```bash
npm install
```

2. Run Playwright tests (chromium):

```bash
npx playwright test tests/e2e/a11y.spec.ts --project=chromium
npx playwright test visual.spec.ts --project=chromium
```
