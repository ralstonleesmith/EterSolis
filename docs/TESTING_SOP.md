# Testing SOP

## Required Gates

- `npm run test:backend`
- `npm run migrations:check`
- `npm run typecheck`
- `npm run lint`
- `npm run docs:check`
- `npm run routes:check`
- `npm run theme:audit`
- `npm run asset:audit`
- `npm run link:audit`
- `npm run build`
- `npm run test:smoke`
- `npm run test:layout`
- `npm run preview:capture`

## Portal Coverage

Tests must cover portal routes, admin case detail, quote-before-invoice copy, payment reconciliation copy, v1 envelope routes, migration presence and generated preview coverage.
